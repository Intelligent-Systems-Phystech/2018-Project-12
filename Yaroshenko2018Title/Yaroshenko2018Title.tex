\abstract
    {В данной работе исследуется задача перевода между двумя языками. Хотя для решения часто используются параллельные предложения, для многих пар языков нет необходимого объема таких фраз, поэтому необходимо использовать методы других форматов. В работе рассматривается одна из альтернативных моделей : она состоит из двух нейросетей, одна из которых отображает предложения из первого языка в скрытое пространство и далее во второй язык, вторая делает это в обратном направлении. [Тут добавится что-то от меня]. Для проверки качества модели проводится вычислительный эксперимент по переводу предложений между близкими языками, такими как русский и украинский.
