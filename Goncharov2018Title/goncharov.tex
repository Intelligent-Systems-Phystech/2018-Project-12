\abstract{
    В данной работе исследуется метод обучения без учителя для решения задачи машинного перевода, предложенный G. Lample и др. в \cite{lample2017unsupervised}. Метод основан на модели seq2seq. Кодировщик обучается отображать предложение в вектор латентного пространства, а декодировщик обучается, используя этот вектор, восстанавливать исходное предложение на любом из двух языков. Процесс обучения построен таким образом, чтобы избежать необходимости использовать параллельные данные. Метод тестируется на паре языков русский-украинский и дает какое-то качество.
    
   \bigskip
   \textbf{Ключевые слова}: \emph {машинный перевод, модель seq2seq, обучение без учителя}.
}

\abstract{
    В последние годы с помощью нейросетевых методов были достигнуты существенные результаты в решении задачи машинного перевода. До недавнего времени все методы были применимы при наличии большого числа параллельных данных. В этом году был предложен метод, в котором процедура оптимизации параметров нейросети использует только данные из произвольных корпусов текста на каждом из языков. В данной работе проверяется качество работы этого метода на паре языков русский-украинский.
    
   \bigskip
   \textbf{Ключевые слова}: \emph {нейросетевой машинный перевод, обучение без учителя}.
}
